# YouTube 同期プレイヤー 引継書

## プロジェクト概要

2つのYouTube動画を左右に並べて、同時再生・一時停止・音量個別調整ができるWebページ。
子ども向けの使いやすいUIにする。

---

## 成果物

単一の HTML ファイル（`youtube-sync-player.html`）として納品。
外部フレームワーク不要。YouTube IFrame Player API のみ使用。

---

## 機能仕様

### URL入力
- フル URL（`https://www.youtube.com/watch?v=XXXXX`）に対応
- 短縮 URL（`https://youtu.be/XXXXX`）に対応
- 埋め込み URL（`https://www.youtube.com/embed/XXXXX`）に対応
- 動画IDを正規表現で自動抽出する

```js
// 抽出ロジック（参考実装）
function extractVideoId(url) {
  const patterns = [
    /[?&]v=([^&#]+)/,
    /youtu\.be\/([^?&#]+)/,
    /embed\/([^?&#]+)/
  ];
  for (const pattern of patterns) {
    const match = url.match(pattern);
    if (match) return match[1];
  }
  return null;
}
```

### プレイヤー初期化
- YouTube IFrame Player API（`https://www.youtube.com/iframe_api`）を使用
- 両プレイヤーが `onReady` 状態になるまで「さいせいスタート」ボタンを非活性にする
- 両方の準備が整ったらボタンを活性化し、自動的に同時再生を開始する

### 同時再生・一時停止
- 「いっしょにさいせい」ボタン → 両プレイヤーで `playVideo()` を同時呼び出し
- 「いっしょにとめる」ボタン → 両プレイヤーで `pauseVideo()` を同時呼び出し
- 再生中は「とめる」ボタンのみ表示、停止中は「さいせい」ボタンのみ表示（トグル）

### シークバー同期
- `<input type="range">` でシークバーを実装
- 操作時は両プレイヤーで `seekTo(seconds, true)` を同時呼び出し
- `setInterval`（250ms）で再生位置を取得し、シークバーの表示を更新
- 長さ（duration）は両動画の短い方を max 値に設定する

### 音量個別調整
- 各プレイヤーの下に独立した音量スライダー（0〜100）
- 操作時に対応するプレイヤーのみ `setVolume(value)` を呼び出す
- デフォルト値は 80

## デザイン仕様（子ども向け）

### 基本方針
- 操作ボタンは大きく・タップしやすく（最小44px以上）
- テキストはひらがな・漢字を最小限に
- 色は明るく、アイコンを併用してわかりやすく

### ボタンサイズ
- メインコントロール（さいせい・とめる）: `font-size: 1.2rem`, `padding: 14px 32px`
- 角丸を大きめに: `border-radius: 999px`（ピル形状）

### カラー
- 「さいせい」ボタン: 緑系（例: `#22c55e`）
- 「とめる」ボタン: オレンジ系（例: `#f97316`）
- 背景: 白または薄いグレー
- プレイヤー枠: 角丸（`border-radius: 12px`）、軽いシャドウ

### レスポンシブ
- 横幅 768px 以上: 左右2カラム
- 横幅 767px 以下: 上下1カラム（モバイル縦持ち対応）

## 実装の注意点

1. **YouTube API の非同期ロード**: `window.onYouTubeIframeAPIReady` コールバックを使う。`YT` オブジェクトはAPIロード完了後にしか使えない。
2. **両プレイヤーの準備待ち**: `player1Ready` と `player2Ready` のフラグを管理し、両方 `true` になったら再生開始可能にする。
3. **シークバーのドラッグ中は更新停止**: ドラッグ中に setInterval が上書きしないよう `isSeeking` フラグを使う。
4. **autoplay パラメータ**: `playerVars: { autoplay: 0 }` にしておき、ボタン押下で再生する（ブラウザのautoplay制限を回避するため）。
